# 开目小沐

开目小沐是 KM-MOM 系统内置的智能 AI 助手，基于大语言模型技术，旨在为用户提供即时的知识问答和操作引导服务。

## 1. 核心功能

### 1.1 智能问答 (Knowledge Q&A)
*   **知识库检索**: 基于企业内部的工艺文档、操作手册和质量标准，提供精准的问答服务。
*   **自然语言交互**: 支持用户使用自然语言提问，例如：“如何创建一个返修工单？”或“查看热处理工艺参数标准”。
*   **上下文理解**: 能够理解多轮对话的上下文，提供连贯的解答。

### 1.2 操作引导 (Operational Guidance)
*   **菜单导航**: 针对新用户，小沐可以根据指令自动跳转到相应的功能菜单。
*   **简单操作代理**: 支持通过语音或文字指令完成简单的系统操作，如“查询今日未完成的检验任务”。
*   **异常辅助**: 在用户遇到系统报错时，自动分析错误原因并给出解决建议。

## 2. 使用场景
*   **新员工培训**: 随时解答系统操作疑问，降低培训成本。
*   **现场作业支持**: 车间工人在操作终端遇到问题时，通过语音快速获取帮助。
