# APS数据模型

## 文档说明

**基本信息**
- 文档版本：v1.1 | 更新日期：2025-12-29 | 维护团队：产品研发团队
- 目标受众：产品研发团队

**文档定位**

本文档定义KMMOM3.x高级计划与排程（APS）模块的数据模型设计，包括工作日历、资源管理、排产方案、分派任务等核心业务实体，为制造执行服务提供排程计算能力。

**内容结构**

| 章节 | 核心问题 | 内容说明 |
|------|---------|----------|
| 一、术语、定义和缩略语 | APS模块涉及哪些专业术语？ | 定义日历、资源、排程等业务概念及枚举值 |
| 二、数据模型 | APS模块包含哪些数据模型？ | 定义工作日历、资源、排产方案、分派任务等12个核心模型 |

---

## 一、术语、定义和缩略语

| 术语 | 定义 | 缩略语 |
|------|------|--------|
| 高级计划与排程 | 基于有限产能和约束条件的生产计划优化与排程计算 | APS |
| 工作日历 | 定义组织或资源的工作时间、休息日、出勤模式的时间管理体系 | - |
| 资源 | 参与生产排程的基本单元，包括工作中心、设备、人员等 | - |
| 资源能力 | 资源针对特定工序的加工能力定义，包括前置、制造、后置时间 | - |
| 分派任务 | 经过排程计算后，分配了资源和时间的工序任务 | - |
| 有限产能 | 考虑资源产能约束的排程方法，任务不能超出资源可用时间 | - |
| 无限产能 | 不考虑资源产能约束的排程方法，允许任务堆叠 | - |
| 正向排程 | 从计划开始时间向后推算任务完成时间的排程方式 | - |
| 逆向排程 | 从交货期向前倒推任务开始时间的排程方式 | - |
| 资源负荷率 | 资源在特定时间段内需求工时与理论产能的比率 | - |

## 二、数据模型

### 2.1 出勤模式（WorkShift）

**模型类型:** 业务模型 | **父模型:** `BusinessObject` | **接口:** `FactoryManaged` | **表名:** `MOM_WORK_SHIFT`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 工作时段 | `timeSlot` | `CTIME_SLOT` | 字符串 | 1024 | 无 | 为空则表示该天休息。最小开始时间和最大结束时间不超过24h，示例：09:00-12:00;14:00-17:00;19:00-21:00; |

### 2.2 工作日历（WorkCalendar）

**模型类型:** 业务模型 | **父模型:** `BusinessObject` | **接口:** `FactoryManaged` | **表名:** `MOM_WORK_CALENDAR`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 开始日期 | `beginDate` | `CBEGIN_DATE` | 日期 | - | 必填 | 日历生效开始日期 |
| 结束日期 | `endDate` | `CEND_DATE` | 日期 | - | 必填 | 日历生效结束日期 |
| 工作日 | `workWeek` | `CWORK_WEEK` | 字符串 | 128 | 无 | 示例:1,2,3,4,5,6,7，用英文逗号分隔 |
| 出勤模式 | `workShift` | `CWORK_SHIFT` | 引用对象 | - | 无 | 引用WorkShift |
| 日历分类 | `calendarCategory` | `CCALENDAR_CATEGORY` | 枚举 | 128 | 必填 | 引用枚举:CalendarCategory |
| 休息标记 | `restFlag` | `CREST_FLAG` | 布尔 | - | 必填 | 默认false，true表示该日历休息，false表示该日历用于出勤 |
| 优先级 | `priority` | `CPRIORITY` | 整型 | - | 必填 | 默认50 |

| ~~资源类型~~ | ~~`resType`~~ | ~~`CRES_TYPE`~~ | ~~枚举~~ | ~~256~~ | ~~必填~~ | ~~资源:RESOURCE~~（此属性无用） |
| 资源对象 | `res` | `CRES` | 引用对象 | - | 必填 | 引用Resource |

### 2.4 资源（Resource）

**模型类型:** 业务模型 | **父模型:** `BusinessObject` | **接口:** `FactoryManaged` | **表名:** `MOM_RESOURCE`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 资源分类 | `resCategory` | `CRES_CATEGORY` | 枚举 | 256 | 必填 | 引用枚举:ResCategory |
| 资源来源ID | `resSourceId` | `CRES_SOURCE_ID` | 长整型 | - | 无 | 记录来源的id |
| 资源组标记 | `resGroupFlag` | `CRES_GROUP_FLAG` | 布尔 | - | 必填 | 标记当前资源是否为资源组，只有为资源组的资源才能添加成员 |
| 所属资源组 | `belongResGroup` | `CBELONG_RES_GROUP` | 引用对象 | - | 无 | 引用Resource |
| 资源量 | `resQty` | `CRES_QTY` | 整型 | - | 必填 | 默认为1，表示资源的份数 |
| 制造效率 | `makeRate` | `CMAKE_RATE` | 浮点型 | - | 必填 | 默认100，表示资源制造期间效率（百分比） |
| 资源制约标记 | `constraintFlag` | `CCONSTRAINT_FLAG` | 布尔 | - | 必填 | 默认为true，false表示不制约，true表示制约 |
| 允许中断标记 | `allowBreakFlag` | `CALLOW_BREAK_FLAG` | 布尔 | - | 必填 | 默认为true，false表示不允许中断，true表示允许中断 |
| 是否有效标记 | `validFlag` | `CVALID_FLAG` | 布尔 | - | 必填 | 默认为false，false表示无效，true表示有效 |
| 所属工厂 | `factory` | `CFACTORY` | 引用对象 | - | 必填 | 引用Org（组织） |
| 所属组织 | `bizOrg` | `CBIZ_ORG` | 引用对象 | - | 必填 | 引用Org（组织） |

### 2.5 资源能力（ResourceCapacity）

**模型类型:** 业务模型 | **父模型:** `BaseObject` | **接口:** `FactoryManaged` | **表名:** `MOM_RES_CAPACITY`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 关联工艺路线 | `routing` | `CROUTING` | 引用对象 | - | 必填 | 针对关联的工艺路线，维护其工序的资源加工能力，引用Routing |
| 关联工序 | `routingProcess` | `CROUTING_PROC` | 引用对象 | - | 必填 | 针对关联的工艺路线的工序，进行资源加工能力维护，引用RoutingProcessLink |
| 使用资源 | `useMainResource` | `CUSE_MAIN_RESOURCE` | 引用对象 | - | 必填 | 针对指定工序，使用资源，引用Resource |
| 前置时间 | `setupTime` | `CSETUP_TIME` | 字符串 | 256 | 必填 | 工序前准备时间 |
| 前置时间单位 | `setupTimeUnit` | `CSETUP_TIME_UNIT` | 单位 | - | 必填 | 引用单位:ProductivityUnit |
| 制造时间 | `makeTime` | `CMAKE_TIME` | 字符串 | 256 | 必填 | 制造时间，不允许为0s |
| 制造时间单位 | `makeTimeUnit` | `CMAKE_TIME_UNIT` | 单位 | - | 必填 | 引用单位:ProductivityUnit |
| 后置时间 | `teardownTime` | `CTEARDOWN_TIME` | 字符串 | 256 | 必填 | 工序后清理时间 |
| 后置时间单位 | `teardownTimeUnit` | `CTEARDOWN_TIME_UNIT` | 单位 | - | 必填 | 引用单位:ProductivityUnit |
| 优先级 | `priority` | `CPRIORITY` | 整型 | - | 必填 | 同工序下资源优先级，默认50 |
| 所属工厂 | `factory` | `CFACTORY` | 引用对象 | - | 必填 | 引用Org（组织） |
| 所属组织 | `bizOrg` | `CBIZ_ORG` | 引用对象 | - | 必填 | 引用Org（组织） |

### 2.6 排序规则定义（SortRule）

**模型类型:** 业务模型 | **父模型:** `BusinessObject` | **接口:** 无 | **表名:** `MOM_SORT_RULE`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 规则表达式 | `ruleExp` | `CRULE_EXP` | 字符串 | 512 | 无 | 排序规则表达式 |
| 系统标记 | `sysFlag` | `CSYS_FLAG` | 布尔 | - | 必填 | true：系统内置；false：业务扩展 |

### 2.7 排产方案（ScheduleSchema）

**模型类型:** 业务模型 | **父模型:** `BusinessObject` | **接口:** `FactoryManaged` | **表名:** `MOM_SCHEDULE_SCHEMA`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 分派规则 | `sortRules` | `CSORT_RULES` | 字符串 | 1024 | 无 | 分派规则的配置使用，格式为分派规则编码#顺序;多个用分号分隔。例如：订单优先级#DESC;订单交货期#ASC; |
| 分派方法 | `assignMethod` | `CASSIGN_METHOD` | 枚举 | - | 必填 | 引用枚举:AssignMethod |
| 分派方向 | `assignDirection` | `CASSIGN_DIRECTION` | 枚举 | - | 必填 | 引用枚举:AssignDirection |
| 分派资源策略 | `assignStrategy` | `CASSIGN_STRATEGY` | 枚举 | 1024 | 必填 | 引用枚举:AssignStrategy |
| 所属工厂 | `factory` | `CFACTORY` | 引用对象 | - | 必填 | 引用Org（组织） |
| 所属组织 | `bizOrg` | `CBIZ_ORG` | 引用对象 | - | 必填 | 引用Org（组织） |

### 2.8 分派任务（AssignedTask）

**模型类型:** 业务模型 | **父模型:** `BaseObject` | **接口:** `FactoryManaged` | **表名:** `MOM_ASSIGNED_TASK`

**说明**：此表为计划域工序任务的扩展（两者主键CID一致），专职处理分派相关数据。

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 关联工序 | `routingProc` | `CROUTING_PROC` | 引用对象 | - | 必填 | 对应关联的工艺路线的工序，引用RoutingProcessLink |
| 关联工序号 | `routingProcCode` | `CROUTING_PROC_CODE` | 字符串 | 128 | 必填 | 对应关联的工艺路线的工序号，冗余属性 |
| 订单ID | `orderId` | `CORDER_ID` | 长整型 | - | 必填 | 所属订单ID |
| 订单号 | `orderCode` | `CORDER_CODE` | 字符串 | 128 | 必填 | 所属订单号，冗余属性 |
| 计划数量 | `plannedQty` | `CPLANNED_QTY` | 浮点型 | - | 无 | 本次分派的计划数量 |
| 本次分派开始时间 | `curAssignedBeginTime` | `CCUR_ASSIGNED_BEGIN_TIME` | 日期时间 | - | 必填 | 记录本次分派的开始时间 |
| 本次分派结束时间 | `curAssignedEndTime` | `CCUR_ASSIGNED_END_TIME` | 日期时间 | - | 必填 | 记录本次分派的结束时间 |
| 本次分派资源 | `curAssignedRes` | `CCUR_ASSIGNED_RES` | 引用对象 | - | 必填 | 记录本次分派的资源，引用Resource |
| 本次分派时间槽 | `curAssignedTimeSlots` | `CCUR_ASSIGNED_TIME_SLOTS` | 字符串 | 4096 | 无 | 表示本次分派时间段，格式如下：20241201090000-20241201123000;20241201140000-20241201183000; |
| 本次分派方向 | `curAssignedDirection` | `CCUR_ASSIGNED_DIRECTION` | 枚举 | - | 无 | 引用枚举:AssignDirection |
| 本次分派方法 | `curAssignedMethod` | `CCUR_ASSIGNED_METHOD` | 枚举 | - | 无 | 引用枚举:AssignMethod |
| 本次分派说明 | `curAssignedExplain` | `CCUR_ASSIGNED_EXPLAIN` | 字符串 | 2048 | 无 | 分派说明信息 |
| 分派总时长 | `assignedTotalDuration` | `CASSIGNED_TOTAL_DURATION` | 整型 | - | 无 | 单位秒，无则为0 |
| 锁定模式 | `lockModel` | `CLOCK_MODEL` | 枚举 | - | 无 | 引用枚举:LockModel |
| 实际开始时间 | `actualStartTime` | `CACTUAL_START_TIME` | 日期时间 | - | 无 | 任务实际开始执行时间 |
| 实际结束时间 | `actualEndTime` | `CACTUAL_END_TIME` | 日期时间 | - | 无 | 任务实际完成时间 |
| 序号 | `index` | `CINDEX` | 整型 | - | 无 | 任务排序序号 |

### 2.9 计划评估（PlanAssess）

**模型类型:** 业务模型 | **父模型:** `BaseObject` | **接口:** `FactoryManaged` | **表名:** `MOM_PLAN_ASSESS`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 排产时间 | `scheduleTime` | `CSCHEDULE_TIME` | 日期时间 | - | 必填 | 记录排产时间 |
| 排产方案 | `scheduleSchema` | `CSCHEDULE_SCHEMA` | 引用对象 | - | 必填 | 记录当时排产使用的排产方案，引用ScheduleSchema |

### 2.10 资源负荷率（ResourceLoadRate）

**模型类型:** 业务模型 | **父模型:** `BaseObject` | **接口:** `FactoryManaged` | **表名:** `MOM_RES_LOAD_RATE`

| 属性中文名称 | 属性英文名称 | 数据库列名 | 数据类型 | 长度 | 约束 | 说明 |
|-------------|-------------|-----------|---------|------|------|------|
| 资源 | `resource` | `CRESOURCE` | 引用对象 | - | 必填 | 对应的资源，引用Resource |
| 计算日 | `calcDay` | `CCALC_DAY` | 日期 | - | 必填 | 表示负荷率计算的哪一天 |
| 计算年份 | `calcYear` | `CCALC_YEAR` | 整型 | - | 必填 | 表示计算日所属年份 |
| 计算年月 | `calcYearMonth` | `CCALC_YEAR_MONTH` | 字符串 | 8 | 必填 | 表示计算日所属年月，例如2024-12 |

| 理论产能 | `theoryCapacity` | `CTHEORY_CAPACITY` | 整型 | - | 必填 | 表示计算日的理论时间，单位秒，由日历出勤所得 |
| 本次负荷率 | `curLoadRate` | `CCUR_LOAD_RATE` | 浮点型 | - | 必填 | 记录本次分派的负荷率（百分比） |
